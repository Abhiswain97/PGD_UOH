y=mx+c

The equation given is the equation for a straight line.

  • m -> slope

  • c -> y-intercept

  • x -> x-coordinate.

Points to be noted

  1. Slope is same everywhere on a line

  2. Two parellel lines have the same slope

  3. Intersecting lines have different slopes

General form of the equation of a straight line

ax+by+c=0

Rearranging to get it into the slope - intercept form gives:

y=−c−axb=−abx+−cb

Here, m = −ab & c=−cb

If we generalize the equation to n dimensions

w1x1+w2x2+⋯+w0=0

x1,x2,…

w1,w2,…

Equation of a plane

π:ax+by+cz+d=0

L:ax+by+c=0

π is used to represent plane

We can also write it as a more general notation:

π:w1x1+w2x2+w3x3+w0=0

Equation of a hyperplane

When the number of dimensions is greater than 2, then we call it hyper-plane It is represented by: πd

It's equation is:

πd:w1x1+w2x2+⋯+wdxd+w0=0

Vectors in Linear Algebra

Below picture is what a vector is. It has a magnitude and a direction.

The magnitude of the above vector is calculated by: ||u→||=a2+b2. This is a 2-d vector.

For a 'd' dimensional vector the formula is generalized to: ||x→||=x12+x22+⋯+xd2

The equation of a hyperplane: πd:w1x1+w2x2+⋯+wdxd+w0=0

can be rewritten in a matrix form using Linear algebra:

[w1…wd] [x1⋮xd]+w0=0

W∈Rd, which means W is a d-dimensional vector and all it's values are real numbers. w0 is a scalar.

The Vector with all w s is a row vector & the vector with all x s is a column vector In maths, when we say vector we mean a column vector by convention

We can write this in a more concise way: WTx+w0=0
